### work-orders-overdue-by-asset

Question: Show overdue work orders by asset.
Problems: Returned columns did not include any of: serial_number, work_order, due_date, status.

```sql
WITH overdue_work_orders AS (
  SELECT
    wo.work_order_number,
    wo.work_date,
    wo.due_date,
    wo.order_status,
    wo.service_status,
    wo.priority,
    wo.equipment_type,
    wo.equipment_model_name,
    wo.equipment_serial_number,
    wo.unit_id AS equipment_id,
    wo.unit_label,
    wo.customer_name,
    wo.site_name,
    wo.site_address,
    (CURRENT_DATE - wo.due_date) AS days_overdue
  FROM work_orders wo
  WHERE wo.due_date IS NOT NULL
    AND wo.due_date < CURRENT_DATE
    AND COALESCE(wo.closed_at, wo.completed_at) IS NULL
)
SELECT
  equipment_id,
  equipment_serial_number,
  equipment_model_name,
  equipment_type,
  COUNT(*) AS overdue_work_order_count,
  MIN(days_overdue) AS min_days_overdue,
  MAX(days_overdue) AS max_days_overdue,
  STRING_AGG(work_order_number, ', ' ORDER BY due_date ASC) AS work_order_numbers
FROM overdue_work_orders
GROUP BY equipment_id, equipment_serial_number, equipment_model_name, equipment_type
ORDER BY overdue_work_order_count DESC, max_days_overdue DESC, equipment_id NULLS LAST
LIMIT 1000
```
